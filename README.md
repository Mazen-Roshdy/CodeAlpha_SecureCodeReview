# Secure Coding Review – CodeAlpha Internship Task

This project demonstrates a secure code review of a vulnerable Java application.

## Objectives

- Identify security vulnerabilities in source code
- Perform manual secure code inspection
- Use a static analysis tool (Semgrep)
- Provide remediation recommendations

## Vulnerabilities Identified

1. Hardcoded database credentials
2. SQL Injection vulnerability
3. Information disclosure via console output
4. Stack trace exposure

## Static Analysis Tool Used

Semgrep

## Files Included

- VulnerableApp.java
- Secure_Code_Review_Report.pdf
- semgrep_report.txt
